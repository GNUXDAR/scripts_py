Referencia del Lenguaje Python¶

Este manual de referencia describe la sintaxis y la «semántica base» del lenguaje. Es conciso, pero intenta ser exacto y completo. La semántica de los tipos de objetos integrados no esenciales y de las funciones y módulos integrados están descritos en La Biblioteca Estándar de Python. Para obtener una introducción informal al lenguaje, consulte El tutorial de Python. Para programadores C o C++, existen dos manuales adicionales: Ampliación e incrustación del intérprete de Python describe detalladamente cómo escribir un módulo de extensión de Python, y Manual de referencia de la API en C de Python describe en detalle las interfaces disponibles para los programadores C/C++.
